# Git 标签同步模块

功能概述：强制同步本地 Git 标签与远程仓库一致。该操作会先删除所有本地标签，然后从远程仓库重新拉取，确保本地环境与远程完全同步。

## 功能特性

-   **全量清理**：自动检测并删除所有本地存在的标签，清除本地可能存在的陈旧或多余标签。
-   **远程同步**：执行 `git fetch --tags` 从远程获取最新标签列表。
-   **一致性保证**：快速解决本地标签混乱或与远程不一致的问题。

## 使用方法

```bash
$ mycli git tag sync
```

## 选项说明

该模块不接受任何命令行选项。

## 注意事项

-   **数据丢失风险**：此命令会**删除所有本地标签**。如果你创建了本地标签但尚未推送到远程仓库，执行此命令后这些标签将**永久丢失**。
-   **前置检查**：请在执行前确保所有需要保留的本地标签都已推送到远程 (`git push origin <tagname>` 或 `git push origin --tags`)。
-   **网络依赖**：拉取过程需要网络连接以访问远程仓库。
