## 通用

- 切换至下一个页面：ctrl + page down
- 切换至上一个页面：ctrl + page up
- 窗口最大化：windows + up
- 关闭窗口：shift + ctrl + w
- 主副屏切换：shift + window + right
- 截图：ctrl + a

## VSCode

- 新窗口打开项目：ctrl + r
- 聚焦编辑器：ctrl + page down
- 聚焦文件资源管理器（自定义）：alt + `
- 从文件资源管理器打开文件：space
- 从文件资源管理器新建文件：ctrl + alt + n
- 从文件资源管理器新建文件夹：ctrl + shift + alt + n
- 将编辑器移动到下一个编辑器组：ctrl + alt + right
- 拆分编辑器：ctrl + \
- 聚焦终端：ctrl + j
- 新建终端：ctrl + shift + `
- 清除终端内容：ctrl + k

## Chrome

- 新建标签：ctrl + t
