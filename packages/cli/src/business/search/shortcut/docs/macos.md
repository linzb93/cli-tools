## 通用

- 切换至下一个页面：shift + cmd + ]
- 切换至上一个页面：shift + cmd + [
- 窗口最大化：无
- 关闭窗口：shift + cmd + w
- 主副屏切换：无
- 截图：shift + command + 2

## VSCode

- 新窗口打开项目：ctrl + r
- 聚焦编辑器：shift + cmd + ]
- 聚焦文件资源管理器（自定义）：option + `
- 从文件资源管理器打开文件：space
- 从文件资源管理器新建文件：ctrl + option + n（自定义）
- 从文件资源管理器新建文件夹：ctrl + option + cmd + n（自定义）
- 拆分编辑器：cmd + \
- 聚焦终端：control + `
- 新建终端：shift + ctrl + `
- 清除终端内容：cmd + k

## Chrome

- 新建标签：command + t
