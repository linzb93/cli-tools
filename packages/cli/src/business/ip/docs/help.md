# IP信息查询模块

IP信息查询工具，提供IP归属地查询、网站IP端口查询、本地IP获取等功能。

## 功能特性

- **IP归属地查询** - 查询指定IP地址的地理位置和运营商信息
- **网站IP和端口查询** - 获取指定网站的IP地址和端口信息  
- **本地IP信息获取** - 获取本机的内网IP和公网IP地址

## 使用方法

```bash
$ mycli ip [ip地址|网站域名]
```

### 查询本地IP信息
```bash
$ mycli ip
```

### 查询IP归属地
```bash
$ mycli ip 8.8.8.8
```

### 查询网站IP和端口
```bash
$ mycli ip google.com
```

## 参数说明

### 位置参数

#### ip地址|网站域名
要查询的IP地址或网站域名。

- **IP地址格式**: 支持标准IPv4地址格式，如 `192.168.1.1`
- **网站域名**: 支持任意网站域名，如 `google.com`、`baidu.com`
- **不提供参数**: 获取本地IP信息（内网IP和公网IP）

类型：`string`（可选）

## 输出格式

### IP归属地查询结果
以彩色表格形式展示：
- IP地址（绿色）
- 归属地信息（绿色）
- 运营商信息（绿色）

### 网站IP查询结果
- IP地址列表
- 端口信息（通常为443）

### 本地IP查询结果
- 内网IP（橙色高亮）
- 公网IP（橙色高亮）

## 技术实现

### 数据来源
- IP归属地：通过ip138.com网站获取
- 公网IP：使用`public-ip`库获取
- 内网IP：使用`internal-ip`库获取
- 网站IP：通过DNS解析获取

### 网络配置
- 请求超时：10秒
- 用户代理：模拟浏览器请求头

## 注意事项

- IP归属地查询依赖第三方网站，可能存在访问限制
- 网站IP查询使用HTTPS端口（443）进行测试
- 公网IPv6地址获取时间较长，当前版本暂不支持
- 查询失败时会显示具体的错误信息
