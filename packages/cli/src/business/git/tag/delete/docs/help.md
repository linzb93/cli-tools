# Git 标签删除模块

功能概述：批量删除本地和远程的 Git 标签。该模块提供交互式界面，允许用户选择多个标签进行删除。

## 功能特性

-   **交互式选择**：提供复选框界面，方便用户批量选择需要删除的标签。
-   **双端删除**：同时删除本地和远程仓库的标签，确保状态同步。
-   **安全检查**：自动检测当前目录是否为 Git 项目，以及是否有可用的标签。

## 使用方法

```bash
$ mycli git tag delete
```

## 交互流程

1.  **环境检查**：命令启动后，首先检查当前目录是否为 Git 项目。如果不满足条件，将输出错误信息并退出。
2.  **获取标签**：扫描当前项目的所有标签。如果未发现标签，将提示警告并退出。
3.  **选择标签**：显示所有标签的列表。用户可以通过键盘操作（`空格键`选择/取消选择，`回车键`确认）来选择一个或多个要删除的标签。
4.  **执行删除**：确认选择后，程序将遍历选中的标签，依次执行本地删除和远程删除操作。

## 选项说明

本模块主要通过交互式界面进行操作，暂无特定的命令行参数或选项。

## 注意事项

-   **权限要求**：请确保您拥有远程仓库的推送权限，以便能够删除远程标签。
-   **操作不可逆**：标签删除后无法直接恢复，请在操作前仔细核对选中的标签。
-   **空选择处理**：如果在交互界面未选择任何标签直接回车，操作将自动取消。
