# Idea Recorder (功能想法记录器)

快速记录开发过程中的功能灵感，并在统一的文档中查阅。

## 功能特性

- **快速记录**: 通过命令行参数或直接读取剪贴板内容，快速保存想法。
- **集中管理**: 所有记录统一保存在项目根目录下的 `docs/ideas.md` 文件中。
- **简单查阅**: 支持通过命令直接查看已记录的想法列表。
- **时间戳**: 每条记录自动附加记录时间。

## 使用方法

### 添加想法

#### 1. 通过参数添加
直接在命令行输入想法内容：
```bash
$ mycli idea add "这是一个很棒的新功能"
# 或者直接作为子命令
$ mycli idea "这是另一个想法"
```

#### 2. 读取剪贴板添加
如果不提供任何参数，将自动读取系统剪贴板中的文本内容进行保存：
```bash
$ mycli idea add
# 或者
$ mycli idea
```

### 查看想法列表

查看所有已记录的想法：
```bash
$ mycli idea list
```

## 数据存储

所有数据保存在项目根目录的 `docs/ideas.md` 文件中。格式如下：

```markdown
# Ideas

- [ ] [2026-02-23 10:00] 这是一个很棒的新功能
- [ ] [2026-02-23 11:30] 这是另一个想法
```

## 注意事项

- 请确保在运行命令时具有对 `docs/ideas.md` 文件的写入权限。
- 如果剪贴板内容为空，将不会记录任何内容。
