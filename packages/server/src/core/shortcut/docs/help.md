# shortcut

`shortcut` 是记录工作中最常用的两个应用（VSCode、Chrome 浏览器）快捷键的命令，根据系统（Windows 系统或 macOS 系统）不同展示不同的内容。

## 使用方法

```bash
$ mycli shortcut [key]
```

查找已记录的快捷键。例如这个是查找已记录的 vscode 快捷键。

```bash
$ mycli shortcut vscode
```

## 参数

### 默认

显示所有已记录的快捷键。

### key

指定要查找的快捷键类型。

-   vscode: VSCode 快捷键
-   chrome: Chrome 浏览器快捷键
-   [key]: 关键字
