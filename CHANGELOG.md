# 更新日志

## 7.0.0 2026-03-06

### 功能

-   不再支持 NodeJS 18 以下的版本，npm依赖全部升级。
-   `git scan`功能支持在终端不用打开项目目录，直接提交和推送代码。
-   `awesome`命令交互由终端改成Web前端，减少终端代码。
-   用python编写Windows右键菜单图像处理的代码。

### bug修复

-   修复`git pull`和`git push`命令在非超时报错时，会重复调用的bug。

### 代码重构

-   接口 router 回调由 Promise 改成使用 async/await。

## 6.0.0 2026-02-17

### 功能

-   tree 命令：新增选项`comment`，含义是添加注释。
-   git scan 命令：支持查看已提交未推送项目的提交详情以及文件。
-   occ 命令添加涨单宝。
-   新增 version 命令，新项目一键创建分支、修改版本等。
-   新增 awesome 命令，用于管理收集的 npm 包，支持增删改查。

### bug 修复

-   git pull 功能修复需要合并时，因为 git 钩子限制，无法提交的 bug。

### 代码重构

-   规范目录结构。
-   多态统一使用工厂模式。
-   业务代码不再使用面向对象，改成函数式，便于测试以及缩小打包大小。
