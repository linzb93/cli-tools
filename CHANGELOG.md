# 更新日志

## 7.0.0 2026-03-11

### 功能

-   awesome 命令：转入Web页面，移除命令行功能。
-   windows系统右键菜单迁入管理。

### 代码重构

-   npm依赖全部升级，不再支持node18以下版本。

## 6.0.0 2026-02-17

### 功能

-   tree 命令：新增选项`comment`，含义是添加注释。
-   git scan 命令：支持查看已提交未推送项目的提交详情以及文件。
-   occ 命令添加涨单宝。
-   新增 version 命令，新项目一键创建分支、修改版本等。
-   新增 awesome 命令，用于管理收集的 npm 包，支持增删改查。

### bug 修复

-   git pull 功能修复需要合并时，因为 git 钩子限制，无法提交的 bug。

### 代码重构

-   规范目录结构。
-   多态统一使用工厂模式。
-   业务代码不再使用面向对象，改成函数式，便于测试以及缩小打包大小。
